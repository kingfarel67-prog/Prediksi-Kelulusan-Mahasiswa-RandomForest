# Prediksi Kelulusan Mahasiswa Menggunakan Random Forest

## Deskripsi

Project ini merupakan implementasi Machine Learning menggunakan metode Random Forest untuk memprediksi apakah mahasiswa akan lulus tepat waktu berdasarkan data akademik.

## Dataset

Dataset berisi informasi mahasiswa:

- Umur
- Jenis Kelamin
- IPS Semester 1
- IPS Semester 2
- Kehadiran
- Jumlah SKS
- Status Bekerja
- Aktivitas Organisasi

Target:

Status Kelulusan

- Lulus Tepat Waktu
- Tidak Tepat Waktu

---

## Algoritma

Random Forest Classifier

Library:

- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

---

## Evaluasi Model

Accuracy : 88%

Precision : 86%

Recall : 84%

F1 Score : 85%
